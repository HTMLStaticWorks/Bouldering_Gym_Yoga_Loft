# Bouldering Gym & Yoga Loft - HTML Template

A premium, multipurpose HTML5 template designed for climbing gyms, indoor bouldering centers, yoga lofts, and wellness studios.

## Features
- **Modern Design**: Athletic and minimal aesthetic with a premium feel.
- **Bootstrap 5**: Built with the latest Bootstrap components.
- **Dark Mode**: Built-in dark mode support with a persistent toggle.
- **RTL Support**: Full support for Right-to-Left languages (Arabic, Hebrew).
- **Responsive**: Mobile-first design for all devices.
- **SEO Optimized**: Semantic HTML5, meta tags, and JSON-LD schema.
- **Animations**: Subtle, high-performance scroll animations.
- **Multipurpose**: Multiple home page layouts and service-focused pages.

## Technologies Used
- HTML5 & CSS3
- JavaScript (ES6)
- Bootstrap 5.3
- Lucide Icons
- Google Fonts (Bebas Neue, Inter)

## Folder Structure
- `assets/css/`: Main styles, dark mode, and RTL support.
- `assets/js/`: Interaction logic and theme toggling.
- `assets/images/`: Placeholder high-quality imagery.
- `pages/`: All template pages (Home, About, Services, Blog, etc.).

## Installation
1. Download the template.
2. Open `pages/index.html` in your browser.
3. To customize, edit the CSS variables in `assets/css/style.css`.
